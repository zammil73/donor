Getting Started

Blood Donation is a complete solution for searching Blood Donors according to location and Blood Group. This CMS Includes almost everything you need to find the right Blood Donor. Blood Donation is fully responsive for any device.

Server Requirements

The Server needs to meet the following requirement to the application.
 
  1. PHP >= 7.2.5
  2. OpenSSL PHP Extension
  3. Mbstring PHP Extension
  4. XML PHP Extension
  5. Fileinfo PHP Extension

Installation

 
After downloading the ZIP archive, and uploading it to your server, the first thing you have to do is to create the database where system tables will be created. Let's say, you create the database called blood_donation.

Step 1 - Welcome Screen

After creating the database next step is accessing the installer from a browser.

To open the Installer. Visit 

yourdomain.com/install
 
Now the installer will show up.The First screen will be Welcome Screen
 

 

Step 2 - Database/Environment Setting

On step 2 you have to fill in your database credentials 

Step 3:Login Credentials

After clicking on "Next" button, you will be redirected to Set Login Credentails step during the installation wizard, Set Login Credentials

Now You are ready to go.
